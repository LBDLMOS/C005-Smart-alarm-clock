# 智能闹钟

#### 介绍
ESP32智能闹钟
在响铃同时移动，并自动避障，用户必须抓到后计算一道数学题方可解除，防止贪睡
软硬件综合项目
支持多组闹钟设定，贪睡模式，避障车模式，强制清醒模式，贪睡模式等

#### 软件架构
C语言
ESP32-IDF开发
